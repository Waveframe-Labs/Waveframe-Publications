# Waveframe Publications

This repository contains publication artifacts produced by Waveframe Labs.

Markdown documents in this repository serve as canonical source files
used by the **Waveframe Forge** to generate Scholar-ready PDF artifacts.

## Purpose

Artifacts in this repository demonstrate practical applications of
Waveframe governance tooling, including deterministic enforcement,
AI-assisted decision accountability, and governed research workflows.

## Build Process

Artifacts are written in Markdown and converted to PDF using the
Waveframe **Forge** GitHub Action.

Workflow:
Markdown source → Forge → Scholar-ready PDF → DOI publication


Generated PDFs are not stored in this repository. The Markdown sources
are the canonical artifacts used to generate publication builds.
